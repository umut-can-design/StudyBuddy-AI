# StudyBuddy AI 🎓🤖
> **Mobil Programlama Dersi Proje Önerisi**

StudyBuddy AI, Android Studio üzerinde geliştirilen, yapay zeka destekli bir kişiselleştirilmiş öğrenme ve odaklanma asistanıdır. Öğrencilerin ders notlarını dijitalleştirmesini, içeriklerden otomatik özet ve test üretmesini sağlar.

## 🌟 Öne Çıkan Özellikler
- **AI Not Analizi:** Kameradan alınan ders notu görüntülerini metne dönüştürür (OCR).
- **Akıllı Özet ve Soru:** NLP kullanarak notlardan saniyeler içinde özet ve çalışma soruları çıkarır.
- **Odak Modu:** Pomodoro tekniği ile çalışma süresini takip eder.
- **Bulut Senkronizasyonu:** Verileri Firebase üzerinde güvenli bir şekilde saklar.

## 🛠 Kullanılan Teknolojiler
- **Dil:** Kotlin / Java
- **Platform:** Android Studio
- **Backend:** Firebase (Auth, Firestore, Storage)
- **AI Engine:** Google Gemini API / OpenAI API
- **ML Kit:** Text Recognition (OCR)

## 📂 Kurulum
1. Bu depoyu klonlayın.
2. Android Studio ile projeyi açın.
3. `local.properties` dosyasına API anahtarlarınızı ekleyin.
4. Firebase `google-services.json` dosyasını `app/` dizinine yerleştirin.
